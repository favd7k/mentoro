# 📊 Mentoro - Executive Summary

## 🎯 Что такое Mentoro?

**Mentoro** - это AI-поддерживаемая платформа управления учебой для студентов, которая объединяет:
- 🤖 Персональный AI-ассистент (24/7)
- 📝 Автоматическое генерирование тестов из учебных материалов
- 📅 Умное управление сроками и расписанием
- 📊 Дашборд с аналитикой прогресса
- 👤 Персональный профиль студента

**Слоган:** "Ваш личный учитель и организатор в одном приложении"

## 💰 Финансовая модель

### Freemium Model
```
Free Tier:
- AI Chat: 5 запросов/день
- Quizzes: 3/месяц
- Календарь: Базовый
- Профиль: Базовый
  → 0₸

Premium ($5-10/месяц):
- AI Chat: Неограниченный
- Quizzes: Неограниченные
- Аналитика: Продвинутая
- Приоритет поддержки
  → $50-100/год
```

### Ожидаемая монетизация
```
Year 1:  10,000 users × 10% × $50/год = $50K
Year 2:  100,000 users × 15% × $50/год = $750K
Year 3:  500,000 users × 20% × $50/год = $5M
```

### B2B (Университеты)
```
Per University License: $1,000-10,000/год
Potential: 50+ университетов × $5,000 = $250K+/год
```
