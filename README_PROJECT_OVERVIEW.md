# 🎓 Mentoro - AI-Powered Academic Assistant Platform

## Быстрый обзор

**Mentoro** - это революционная платформа управления учебой, которая объединяет AI-ассистента, автоматическую генерацию квизов, управление сроками и аналитику - все в одном месте.

### Основная идея в одном предложении:
> "Duolingo для всех предметов, но с собственным AI-учителем"
