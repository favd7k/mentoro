# 🎓 Mentoro - Полная информация для питча

---

## 📌 Краткое резюме

**Mentoro** - это инновационная AI-поддерживаемая платформа для управления академической жизнью студентов. Приложение помогает студентам организовать свои курсы, отслеживать сроки, готовиться к экзаменам через интерактивные квизы и получать персональную помощь от AI-ассистента в реальном времени.

---

## 🎯 Проблема, которую мы решаем

### Боли студентов:
1. **Перегруженность информацией** - Множество курсов, учебников, заданий, сроков
2. **Низкая организация** - Трудно отслеживать все дедлайны и важные события
3. **Недостаток поддержки** - Ограниченный доступ к преподавателям и помощи при возникновении вопросов
4. **Неэффективное обучение** - Нужны способы проверить знания и получить обратную связь
5. **Стресс и выгорание** - Отсутствие инструментов для планирования и организации времени

---

## 💡 Наше решение

**Mentoro** - это все-в-одном платформа, которая:

✅ **Интегрирует все аспекты учебы** в одном месте
✅ **Автоматизирует планирование** и управление временем
✅ **Предоставляет AI-помощника** для ответов на вопросы 24/7
✅ **Генерирует тесты и квизы** из учебного материала автоматически
✅ **Отслеживает прогресс** и мотивирует студентов

## 🔧 Как это работает

### 1️⃣ **Аутентификация**
```
User → Login Page → API /auth/login → JWT Token → Dashboard
```

- Email/пароль аутентификация
- JWT tokens для сессий
- Bcryptjs для шифрования паролей

### 2️⃣ **AI Chat**
```
User Input → /api/ai/chat → LM Studio/OpenAI → Response → Display
```

- Интегрируется с LM Studio для локальных моделей
- Может использовать OpenAI API для более продвинутых моделей
- Real-time streaming ответов

### 3️⃣ **Генерация Квизов**
```
PDF Upload → /api/quizzes/upload → Extract Text → /api/quizzes/generate → AI Questions → Database
```

- Загрузка документов
- Извлечение текста из PDF
- AI генерирует вопросы и ответы
- Сохранение в базу данных

### 4️⃣ **Управление Календарем**
```
Dashboard → Calendar View → Filter by Date → Show Deadlines → Notifications
```

## 💻 Технические детали

### API Endpoints:

```
Authentication:
POST   /api/auth/register       - Регистрация
POST   /api/auth/login         - Вход
GET    /api/auth/me            - Текущий пользователь

AI Chat:
POST   /api/ai/chat            - Общение с AI

Quizzes:
POST   /api/quizzes/upload     - Загрузить документ
POST   /api/quizzes/generate   - Генерировать квиз
POST   /api/quizzes/chat       - Chat по материалу
```

### Базы данных (планируется):
- Supabase PostgreSQL для основных данных
- Redis для кеша и sessions
- S3 для хранения документов

### Масштабируемость:
- Next.js serverless functions
- CDN для статических файлов
- Database query optimization
- Rate limiting для API
