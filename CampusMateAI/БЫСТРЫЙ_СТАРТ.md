# 🚀 Mentoro - Быстрый старт

## Простая инструкция для запуска проекта

### Шаг 1: Откройте терминал
- **Windows**: Нажмите Win + R, введите `cmd`, Enter
- **Mac**: Cmd + Space, введите `terminal`, Enter

### Шаг 2: Перейдите в папку проекта
```bash
cd CampusMateAI
```

### Шаг 3: Установите пакеты
```bash
npm install
```
⏱️ Подождите 2-5 минут

### Шаг 4: Запустите проект
```bash
npm run dev
```

### Шаг 5: Откройте браузер
Перейдите на: **http://localhost:3000**
