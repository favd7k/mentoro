# 🎓 Mentoro - Installation Guide

## 📋 Что вам понадобится

Перед установкой убедитесь, что у вас установлено:
- **Node.js** (версия 16 или выше) - [Скачать здесь](https://nodejs.org/)
- **npm** (устанавливается автоматически с Node.js)

---

## 🚀 Шаги установки

### 1️⃣ Откройте терминал/командную строку

**Windows:**
- Нажмите `Win + R`, введите `cmd`, нажмите Enter

**Mac/Linux:**
- Нажмите `Cmd + Space`, введите `terminal`, нажмите Enter

### 2️⃣ Перейдите в папку проекта

```bash
cd CampusMateAI
```

### 3️⃣ Установите все необходимые пакеты

```bash
npm install
```

Эта команда скачает и установит все зависимости из файла `package.json`. Процесс может занять 2-5 минут.

### 4️⃣ Запустите проект

```bash
npm run dev
```

### 5️⃣ Откройте браузер

Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## 🤖 Настройка AI (опционально)

Для работы AI-чата вам нужно:

1. **Скачайте LM Studio:**
   - Перейдите на [https://lmstudio.ai/](https://lmstudio.ai/)
   - Скачайте и установите для вашей системы

2. **Загрузите модель:**
   - Откройте LM Studio
   - Скачайте любую совместимую модель (например, Llama 2)
   - Запустите локальный сервер на порту 1234

3. **Готово!** AI-чат теперь будет работать.

## ❗ Решение проблем

### Проблема: "Порт 3000 уже занят"
**Решение:**
- Закройте другие приложения, использующие порт 3000
- Или измените порт в файле `package.json`

### Проблема: "npm: command not found"
**Решение:**
- Установите Node.js с [https://nodejs.org/](https://nodejs.org/)
- Перезапустите терминал

### Проблема: Ошибки при установке
**Решение:**
```bash
# Удалите старые файлы
rm -rf node_modules
rm -rf .next

# Переустановите зависимости
npm install
```

### Проблема: AI не отвечает
**Решение:**
- Убедитесь, что LM Studio запущен
- Проверьте, что сервер работает на http://localhost:1234
- Загрузите модель в LM Studio
